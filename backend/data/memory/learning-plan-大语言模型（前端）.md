## 大语言模型（前端） 学习计划
- 目标：掌握大语言模型的前端开发技能
- 总步骤：6步
- 当前进度：0/6
- 创建时间：2026-06-30

### 学习步骤：
1. 第1阶段（1-2周）：LLM基础认知 - 了解大语言模型的基本概念、API调用方式（OpenAI/Claude/通义千问等）、Prompt Engineering基础
2. 第2阶段（3-4周）：LLM API集成 - 学习如何在前端项目中调用LLM API，使用fetch/axios发送请求，处理流式响应（SSE/WebSocket）
3. 第3阶段（5-6周）：对话式UI开发 - 构建类ChatGPT聊天界面，实现消息管理、流式打字效果、Markdown渲染、代码高亮
4. 第4阶段（7-8周）：高级交互模式 - 实现多轮对话上下文管理、函数调用(Function Calling)、Agent交互、知识库检索的前端集成
5. 第5阶段（9-10周）：记忆与状态管理 - 使用LangChain/自建记忆系统、本地存储历史、登录态与用户管理
6. 第6阶段（11-12周）：性能优化与部署 - 流式渲染优化、组件懒加载、SSR/SSG适配、Vercel/云平台部署
