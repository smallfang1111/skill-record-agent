## 雅思 学习计划
- 目标：雅思7分
- 总步骤：8步
- 当前进度：0/8
- 创建时间：2026-06-30

### 学习步骤：
1. 1. 评估当前水平 - 先做一套模拟题，了解自己的强项和弱项（听力、阅读、写作、口语）
2. 2. 词汇积累 - 每天背诵30-50个雅思核心词汇，重点掌握学术词汇和同义词替换
3. 3. 听力训练 - 每天听30分钟BBC/雅思真题，练习精听和泛听结合，重点攻克Section 3&4
4. 4. 阅读提升 - 每天精读1篇雅思阅读文章，练习扫读、定位、同义替换技巧，限时20分钟/篇
5. 5. 写作强化 - 每周写2篇大作文+2篇小作文，掌握Task 1图表描述和Task 2议论文结构
6. 6. 口语练习 - 每天练习10-15分钟口语Part 1&2&3，录音纠正发音和语法错误
7. 7. 真题模考 - 每周至少做1套完整真题，严格计时，模拟真实考试环境
8. 8. 错题分析 - 每套题后认真分析错题原因，总结薄弱题型针对性突破
